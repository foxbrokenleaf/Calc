## 开发环境
+ 操作系统:Windows 11 Homes 23H2
+ 编译器:Keil C51
+ 开发工具:Visual Studio Code + EIDE
+ 烧入工具:CH340
## 使用器件
+ MCU芯片：STC15W408AS-DIP28
+ 显示:LCD1602 5V 黄绿屏
+ 输入:轻触按键*16（矩阵键盘）
+ 背光控制:光敏电阻 + 10kΩ
+ LCD1602对比度：5KΩ电位器
+ 洞洞板
